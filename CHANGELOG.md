# 更新日志

## [2026-02-01] - 移动端UI优化与部署修复

### 修复
- **移除CDN importmap**: 删除 `index.html` 中的 CDN importmap 配置，避免浏览器从 CDN 缓存加载旧版本组件
- **修复部署路径**: 修正前端部署目录从 `/var/www/essay-frontend` 到 `/var/www/essay-grader/frontend`，与 Nginx 配置保持一致

### 优化
- **移除测试功能**: 删除 Header 组件中的"重置次数"测试按钮（桌面端和移动端）
- **移动端UI改进**:
  - 修复标签导航栏在移动端的显示问题
  - 优化按钮布局，防止文字换行
  - 修复滚动问题，确保页面可以正常滚动
  - 缩短按钮文字（如"自动预测题目"→"预测"）以适应小屏幕
